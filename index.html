How to Use Torn Bounty Hunter

API Key

  • Paste your Torn API key into the field at the top and click Save Key.
  • Your key is stored in your browser only — it is never sent to this site's servers. All API calls go directly from your browser to api.torn.com.
  • Use Clear Key to remove the saved key from this device.
  • Generate or find your API key at Torn → Settings → API Key. A Public access level is sufficient.

Auto-Refresh

  • Use the Auto drop-down (Off / 2m / 5m / 10m) to automatically reload bounties at a set interval.
  • A countdown timer (e.g. ↻ 1:45) appears next to the selector while a refresh is scheduled.
  • Clicking Refresh at any time resets the countdown.
  • Your chosen interval is remembered between visits.

Filters

  • Name search — filter by partial player name (case-insensitive).
  • Anonymous — show all bounties, only those listed anonymously, or only named listings.
  • Has reason — show only bounties with (or without) a stated reason.
  • Min / Max level — restrict results to a player-level range.
  • Min quantity — show only bounties with multiple listings (useful for high-value targets).
  • Min reward / Min total value — hide low-paying bounties.
  • Sort by — order by reward, total value, level, or quantity in either direction.
  • All filter settings are saved automatically and restored on your next visit. Use Reset to clear them.
  • On mobile the filter panel is collapsed by default; tap Filters ▾ to expand. The badge shows how many filters are active.

My Level & Fair Fight

  • Enter your own Torn level in the My level field to enable fair-fight highlighting.
  • Any bounty target within ±10 levels of your level is highlighted with a blue border — a rough indicator of a fair-fight range.
  • Note: Torn's true fair-fight calculation is based on battle stats, not level. This highlight is an approximation only.
  • Your level is saved between visits.

Player Status

  • Once you have filtered to 100 or fewer visible bounties, the Check Status button turns green and becomes active.
  • Clicking it fetches a live profile for every visible target and adds an info row to each card showing:
  • Status badge — ● Okay · ✚ Hosp (with time remaining) · ✈ Traveling · ⚑ Jail · ⊘ Federal. Hover for the full description.
  • ⚔ Rank · Title — battle rank (e.g. Reasonable) and honour title (e.g. Slayer), a better indicator of combat strength than level alone.
  • ⏱ Last active — how recently the target was seen online (e.g. "6 minutes ago").
  • ♥ Life % — current life as a percentage of maximum. Green above 66 %, amber below, red below 33 %.
  • ⚕ Revivable — shown only when the target is hospitalised and can be revived.
  • Status is a snapshot — use ↻ Re-check to refresh it. Bounty data refreshes clear status automatically.
  • Location filter — after Check Status, the Location drop-down in the filter bar activates. Select a country to show only targets there, or Torn City for everyone currently at home.

Fair Fight (FFScouter)

  • The Check FF button fetches fair fight values and battle stat estimates from FFScouter for all visible targets.
  • The first time you use it you will be asked to register your Torn API key with FFScouter — a one-time step. Review their data policy at ffscouter.com before agreeing.
  • FF × — the fair fight multiplier applied to your attack reward. Above 1× means a bonus; below 1× means a penalty.
  • ⚡ stat estimate — the target's estimated total battle stats (e.g. 2.99b), a more accurate measure of combat strength than level alone.
  • Like status, FF data is a snapshot — use ↻ Re-check FF to refresh it. Bounty data refreshes clear it automatically.

Card Highlights

  • Red border — bounty expires in under 60 minutes. The expiry time also turns red.
  • Blue border — target is within your fair-fight range (requires My level to be set).
  • The Hunt button on each card opens the Torn bounty page for that player in a new tab, where you can attack and claim the reward.
  • Clicking a player's name opens their Torn profile in a new tab.
  • The ×N badge shows how many open bounty listings exist for that player. The total payout across all listings is shown below the per-listing reward.
